# learning-mentor

Repository containing claude guidelines for generating learning projects,
milestines and reviewing progress.

Just prompt claude with a project description like:

1. Building an in memory cache and different caching strategies
2. Building a blockchain
3. Learning different types of consensus algorithms

Claude will generate:

1. `setup.md` - setup guide for your environment
2. `milestones/` - guides for building out each milestone
3. `real-world-examples.md` - examples of real world usage for your problem
